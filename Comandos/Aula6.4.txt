kubectl get pods
kubectl run --help
kubectl run nginx --generator=run-pod/v1 --image=nginx

kubectl get pods -o wide
kubectl describe pod nginx


Descrição dos comandos

kubectl run: Crie e execute uma imagem específica, possivelmente replicada;
--generator=run-pod/v1: O nome do gerador de API a ser utilizado;
--image: O nome da imagem Docker;
kubectl get pods -o wide: Lista todos os pods no formato de saída ps com mais informações (como nome do nó);
kubectl describe pod: Exibe informações sobre um Pod.
